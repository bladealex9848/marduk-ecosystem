Contribuir al Proyecto

Ayúdanos a mejorar el ecosistema Marduk

¿Por qué contribuir?

El ecosistema Marduk es un proyecto colaborativo que se beneficia de las contribuciones de la comunidad. Tu participación puede ayudar a mejorar el acceso a la justicia y la eficiencia del sistema judicial.

Ver en GitHub

Desarrollo de código

Cómo contribuir con código

Contribuye con nuevas funcionalidades, correcciones de errores o mejoras de rendimiento. Aceptamos pull requests en nuestro repositorio de GitHub.

Pasos para contribuir:
  1. Revisa los issues abiertos o crea uno nuevo
  2. Haz fork del repositorio
  3. Crea una rama para tu contribución
  4. Implementa tus cambios siguiendo las guías de estilo
  5. Envía un pull request
