Pruebas de OpenRouter API

Este directorio contiene varias pruebas para verificar la conexión y funcionalidad de la API de OpenRouter.

Pruebas de Navegador

Estas pruebas se ejecutan en el navegador y son útiles para verificar la integración con la interfaz de usuario.

Acceso a .env

Prueba si el archivo .env es accesible desde el navegador.
Carga Directa de .env

Prueba la carga directa del archivo .env desde JavaScript.
Modelos Mejorados

Muestra los modelos disponibles en OpenRouter, tanto locales como desde la API.
OpenRouter Mejorado

Prueba mejorada de la API de OpenRouter con interfaz de usuario.
Modelos (Original)

Versión original de la lista de modelos.
OpenRouter Seguro (Original)

Versión original de la prueba segura.
Pruebas de Terminal

Estas pruebas se ejecutan desde el terminal y son útiles para verificar la conexión con la API de OpenRouter sin necesidad de un navegador.

test-api-terminal.js

Lee la API key directamente del archivo .env en la raíz del proyecto y prueba la conexión con la API de OpenRouter.

Ejecutar: node tests/test-api-terminal.js
test-terminal-final.js

Versión simplificada que prueba la autenticación y las solicitudes de chat.

Ejecutar: node tests/test-terminal-final.js
run-all-tests.js

Ejecuta todas las pruebas de terminal y muestra un resumen de los resultados.

Ejecutar: node tests/run-all-tests.js
open-browser-tests.js

Abre todas las pruebas de navegador en el navegador predeterminado.

Ejecutar: node tests/open-browser-tests.js
